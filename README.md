# GeoplanDB
Прикладной проект, созданный по ТЗ ныне несуществующей компании ООО ГК "ГЕОПЛАН" для представления в качестве выпускной квалификационной работы факультета СПО "программирование в компьютерных системах".

# Что это?

Проект представляет из себя программу для удобной работы сотрудников геодзической компании с внутренней рабочей базой данных. В основе своей данный проект был мне лично интересен скорее с точки зрения опыта создания рабочего интерфейса в строгом, но, при этом, приятно-выглядящем стиле, чего и удалось добиться:

- Много кофе

# Итог

Суммируя всю проделанную работу и полученный опыт, конечно можно сказать, что работа совсем базовая, однако же свои функции она выполняет, пускай и с реализацией на мобильном устройстве произошли некоторые проблемы, связанные с моей серьёзной неопытностью в момент написания сего проекта (2020-ый год). Ну и конечно же, ужатые сроки тоже сыграли свою роль. На данный момент программа не коннектится к базе данных, потому как её уже попросту нет, в связи с закрытием компании, ОДНАКО есть развёртываемая плейсхолдер-база, с которой, по меньшей мере, программа может работать и сохранять добавленные значения.
